# Appraisal API

This is a sample API for generating appraisal reports for a given property. The API is built using NestJS.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run dev

# production mode
npm run start
```

## Sample .env file

```bash
DATABASE_URL=""
NEXT_AUTH_SECRET=""
OPENAI_API_KEY=""
```
